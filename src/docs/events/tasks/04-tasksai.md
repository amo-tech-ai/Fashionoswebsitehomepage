### üß© Core AI Agents & Workflows

To support these features, the following AI Agents are architected into the system:

1.  **The Orchestrator:** Decomposes briefs into executable work breakdown structures (WBS).
2.  **The Risk Guardian:** Continuously scans the task graph for critical path violations and budget leakage.
3.  **The Visual Asset Mapper:** Ensures every task is grounded in the physical reality of the show (Looks, Venue, Talent).
4.  **The State Sync Engine:** Maintains truth across the disparate "Work Drawers" (Runway, Press, Logistics).
