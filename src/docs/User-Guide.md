## Getting Started

### Creating Your Account

1. Visit `https://fashionos.com`
2. Click **"Start Free Trial"** or **"Sign Up"**
3. Choose your user role:
   - **Event Planner** - Organize fashion events
   - **Designer** - Showcase your collections
   - **Sponsor** - Invest in events and track ROI
   - **Venue Manager** - Manage venue logistics
4. Fill in your details and verify your email
5. Complete your profile setup

### Dashboard Overview

After logging in, you'll see your personalized dashboard based on your role:

**Event Planner Dashboard:**
- Quick access to all your events
- Upcoming tasks and deadlines
- AI-generated insights and alerts
- Recent activity feed

**Designer Dashboard:**
- Your designer profile at a glance
- Event invitations and assignments
- Collection management
- Media library access

**Sponsor Dashboard:**
- Active sponsorships and pipeline
- ROI analytics for past events
- Event discovery and recommendations

## Event Planner Guide

### Creating a New Event

#### Step 1: Start Event Creation
1. Click the **"+ New Event"** button in the sidebar or dashboard
2. The Event Creation Wizard will open

#### Step 2: Basic Information
Fill in the essential event details:
- **Event Name:** e.g., "Paris Fashion Week - Emerging Designers"
- **Event Type:** Runway Show, Trunk Show, Activation, Pop-Up, or Photoshoot
- **Date Range:** Start and end dates
- **Venue:** Select from existing venues or add a new one
- **Budget:** Total event budget
- **Target Attendees:** Expected number of guests

#### Step 3: Review Auto-Generated Tasks
FashionOS automatically creates 127+ tasks across 5 phases:
- **Pre-Production:** Goals, Budget, Team setup
- **Venue & Logistics:** Booking, Vendors, Safety planning
- **Creative Design:** Concept, Assets, Run-of-show
- **On-Site Ops:** Installation, Rehearsals, Show execution
- **Post-Event:** Reporting, ROI, Analytics

Review and customize these tasks as needed.

#### Step 4: Launch Event
Click **"Launch Event"** to create your event and navigate to the Event Command Center.

### Using the Event Command Center

The Command Center is your mission control for managing every aspect of an event.

#### Key Sections:

**1. KPI Dashboard (Top)**
Real-time metrics displayed as cards:
- **Event Progress:** Overall completion percentage
- **Tasks Active:** Number of active tasks (with overdue count)

Visual representation of the 5-phase workflow:

**3. Smart Alerts Panel (Right)**
AI-generated alerts with severity levels:
- **High (Red):** Immediate attention required
- **Medium (Yellow):** Review soon

Click an alert to see details and recommendations.

**4. Quick Navigation Links**
One-click access to all sub-systems:
- Venue Management
- Runway Stage
- Casting & Models
- Designer Collection
- Sponsor CRM
- Tasks & Deliverables
- Contracts

### Managing Tasks & Deliverables

#### Accessing Tasks
1. From Command Center, click **"Tasks & Deliverables"**
2. Or navigate via sidebar → **Tasks**

#### Task Board Overview
Tasks organized by category tabs:
- Event Planning
- Sponsorship
- Marketing
- Operations
- Media

Each category shows a Kanban board with columns:
- **To Do** - Not started
- **In Progress** - Currently being worked on
- **Review** - Awaiting approval
- **Done** - Completed

#### Creating a Task
1. Click **"+ New Task"** button
2. Fill in details:
   - **Title:** Clear, action-oriented (e.g., "Finalize model walk order")
   - **Description:** Additional context
   - **Category:** Select appropriate category
   - **Priority:** Low, Medium, or High
   - **Assigned To:** Team member responsible
   - **Due Date:** Deadline
   - **Dependencies:** Tasks that must be completed first
3. Click **"Create Task"**

#### Updating Task Status
- **Drag and Drop:** Drag task cards between columns
- **Or Click:** Open task details → Change status dropdown

#### Adding Deliverables
1. Open task details
2. Scroll to **"Deliverables"** section
3. Click **"Upload Files"** or drag files into upload area
4. Supported formats: PDF, JPG, PNG, MP4 (max 10MB per file)
5. Files are automatically attached to the task

#### Filtering Tasks
Use filter options at top of task board:
- **By Assignee:** Show only your tasks or specific team member's tasks
- **By Priority:** Filter High/Medium/Low priority
- **Overdue Only:** Show tasks past their due date
- **Search:** Search by task title or description

### Runway Stage Planning

#### Designing Walk Order
1. Navigate to **Runway Stage** from Event Command Center
2. View interactive stage layout
3. Use **Walk Order Sequencer:**
   - Drag model profiles from right panel

4. Click **"Optimize with AI"** to let AI suggest optimal order based on:
   - Wardrobe change times
   - Visual flow (color palettes)
   - Designer preferences

#### Adding Music & Lighting Cues
1. For each look in walk order, expand details

#### Run-of-Show Timeline
1. Click **"Generate Run-of-Show"** button

3. Includes:
   - Guest arrival and seating
   - Show start
   - Each look with timing
   - Designer bows
   - Show end and afterparty start
4. Export as PDF to share with production team

## AI Features Guide

### Understanding AI Insights

FashionOS uses Google Gemini AI to provide intelligent insights across the platform.

#### Where to Find AI Insights
- **Event Command Center:** Right panel labeled "AI Insights"
- **Designer Directory:** "Discovery Insights" panel
- **Sponsor CRM:** "Sponsor Recommendations"
- **ROI Analytics:** "Predictive Insights"

#### Types of AI Insights

**1. Risk Detection & Alerts**
- AI monitors event metrics in real-time
- Flags potential issues before they become problems
- Examples:
  - "Backstage congestion risk detected"
  - "Lighting test overdue"
  - "Budget trending 12% over forecast"
- Each alert includes:
  - Severity level (High/Medium/Low)
  - Clear description
  - Actionable recommendation
  - Related workflow step

**2. Predictive Analytics**
- Forecasts event outcomes based on historical data
- Examples:
  - Estimated attendance with confidence range
  - Predicted social media impressions
  - Budget projection at current burn rate
- Use predictions to set realistic targets and allocate resources

**3. Trend Analysis**
- Identifies emerging patterns in fashion and events
- Examples:
  - "Sustainable streetwear trending +45% this quarter"
  - "Emerging designer 'Kaelo Studios' gaining momentum"
  - "Gen-Z engagement highest at intimate pop-up events"
- Use trends to inform event themes and designer selection

**4. Smart Recommendations**
- AI suggests actions to improve outcomes
- Examples:
  - "Add 2 sustainable designers to attract eco-conscious sponsors"
  - "Increase Instagram budget by 10% for 15% higher reach"
  - "Consider booking 'Grand Palais' for 20% higher media value"

## Tips & Best Practices

### For Event Planners

**1. Start Early**
- Create your event at least 3 months in advance
- This gives AI more time to analyze and provide insights
- Allows buffer time for task completion

**2. Use AI Alerts Proactively**
- Check AI Insights panel daily
- Address medium-severity alerts within 48 hours
- High-severity alerts = immediate action required

**3. Leverage Auto-Generated Tasks**
- Review all auto-generated tasks when creating an event
- Customize as needed, but don't delete entire categories
- Tasks are based on industry best practices

**4. Keep Budget Updated**
- Log all expenses as they occur
- AI uses real-time budget data for forecasting
- Prevents last-minute surprises

**5. Invite Designers Early**
- Send invitations 6-8 weeks before event
- Gives designers time to prepare collections
- Reduces last-minute scrambling

---

**1. Complete Your Full Profile**
- Profiles with all 6 tabs completed get 3x more event invitations
- AI prioritizes complete profiles in recommendations

**2. Upload High-Quality Images**
- Use high-res images (min 2000px width)
- AI image analysis works best with clear, well-lit photos
- Professional photography recommended

**3. Update Collections Seasonally**
- Add new collections every season
- Keep profile fresh and relevant
- Signals to AI that you're active

**5. Use Tags Strategically**
- Add all relevant tags to your profile

2. Check spam folder for verification email
3. Wait 30 minutes if account is locked, then try again

#### "My event isn't generating AI insights"
**Possible causes:**
- Event was just created (AI needs a few minutes to analyze)
- Insufficient data entered (missing budget, attendees, or tasks)
- AI service temporarily unavailable

**Solutions:**
1. Wait 5-10 minutes after creating event
2. Ensure event has: budget, target attendees, and at least 10 tasks
3. Click **"Refresh Insights"** button
4. If still not working, contact support

---

#### "Uploaded images aren't appearing"
**Possible causes:**
- File size too large (max 10MB per image)
- Unsupported file format
- Slow internet connection

**Solutions:**
1. Compress images before uploading (use online tool like TinyPNG)
2. Ensure format is JPG, PNG, or WEBP
3. Check internet connection and retry
4. Try uploading one image at a time

---

#### "Tasks aren't syncing across my team"
**Possible causes:**
- Team members haven't been invited to event
- Browser cache needs clearing
- Task assigned to wrong person

**Solutions:**
1. Verify team members are invited: Event Settings → Team Members
2. Clear browser cache and refresh page
3. Check task assignment: Open task → Verify "Assigned To" field
4. Ask team member to log out and back in

---

#### "Sponsor ROI calculations seem incorrect"
**Possible causes:**
- Missing data (attendance, social metrics, or media coverage)
- External API delays (social media data takes 24-48hrs post-event)
- Manual data entry errors

**Solutions:**
1. Verify all data fields are filled in Analytics Dashboard
2. Wait 48 hours after event for full data collection
3. Click **"Recalculate ROI"** button
4. Contact support if numbers still seem off

---

#### "AI-generated content doesn't match my brand tone"
**Possible causes:**
- Generic brand profile
- AI learning from limited data

**Solutions:**
1. Update your brand profile with more specific tone guidelines
2. Use the **"Tone"** dropdown when generating content (options: Professional, Luxury, Edgy, Playful)
3. Edit AI output to match your voice (AI learns from your edits)
4. Provide more context in generation prompts
